# updater

Monorepo for managing alerts and broadcasting them via UI web components to the element group systems: earth, water, fire, and air.

## Workspace Packages

- `updater-zone`: React + Vite app for CSMs to manage and publish alerts.
- `api`: NestJS backend API.
- `common`: Shared types, enums, and constants.
- `component`: SolidJS web components package.
